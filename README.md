### Instration

1. Clone this reposity.
```
git clone git@github.com:yushimatenjin/typescript-with-playcanvas-editor.git
```

3. Install packages.
```
yarn install
```

4. Generate playcanvas.json.
```
cd typescript-with-playcanvas-editor
yarn playcanvas-cli init
```
